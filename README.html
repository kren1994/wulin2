wulin

ブラウザで使える Renju（連珠）対局所です。合言葉で同じ部屋に入り、2 人で対局、他の参加者は観戦できます。

対局の流れ

  1. 入室する
    表示名と合言葉を入力して入室します。合言葉が同じなら同じ部屋に接続されます。
  2. 席に着く（黒 / 白）
    黒席・白席の + で着席します。席が空いていれば参加者自身で着席できます。
  3. 準備 OK を押す
    着席した 2 人がそれぞれ準備 OK にすると、自動で対局が始まります。
  4. 開局（Taraguchi-10）
    開局は Taraguchi-10 進行です。途中で Swap 選択や、黒の Choice（10 題提示）分岐があります。
  5. 対局を進める
    着手すると持ち時間が加算されます（フィッシャールール）。
  6. 終局・局後検討
    時間切れ、時間内に復帰なし、終局、満局合意で対局終了。終了後は局後検討モードで手順を前後できます。

主な機能

  • 2 人対局 + 観戦席
  • プレイヤー名の末尾に * が表示されている人はホスト
  • ホストは以下の権限をもつ
    • 持ち時間設定（基本分 + 秒加算）
    • 対局席の退席管理
  • チャット（入退室ログ / コメント投稿）
  • 満局提案、待った、終局
  • 局面ナビゲーション（<<, <, 最新, >, >>、矢印キー対応）
  • 局後検討（終局後のみ）
  • 表示設定（言語切替、効果音 ON/OFF、座標表示 ON/OFF、盤面色切替、手数表示 ON/OFF）
  • 再接続対応（切断時は一時停止して復帰待機。同じ表示名・同じ合言葉で復帰でき、一定時間復帰できない場合は終局）
